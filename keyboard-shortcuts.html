Better Shortcuts

  1. Clipboard history : Windows + V
  2. Windows alignment : Windows + right arrow/left arrow/up arrow/down arrow
  3. Taskbar apps start : Windows + 1/2/3/4/5/.....
  4. Minimize/display all windows : Windows + D
  5. Delete full word : Ctrl + Backspace/Delete
  6. Open recent or running windows : Windows + Tab / Alt + Tab
  7. Select full element before/after cursor : Shift + Home/End
  8. Select full word before/after cursor : Ctrl + Shift + left/right arrow
  9. Lock the screen : Windows + L
  10. Open/close any link on new tab : Click on mouse wheel
  11. Open emoji panel : Windows + "." or ";"
  12. Capture a screenshot : Windows + PrtScn

Another Important Shortcuts

  1. Move the cursor to the beginning/end of the line : Home/End
  2. Move the cursor to the beginnign/end of the entire document : Ctrl + Home/End
  3. Create new folder on desktop or file explorer : Ctrl + Shift + N
  4. Close active window : Alt + F4
  5. Delete selected item permanently skipping Recycle bin : Shift + Delete
  6. Rename selected item : F2
  7. Open settings app : Windows + I
  8. Open File Explorer : Windows + E
